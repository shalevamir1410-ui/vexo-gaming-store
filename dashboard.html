הגדרות DSers Webhook

🔗 URL לחיבור DSers:

העתק את ה-URL הזה והדבק אותו בהגדרות ה-Webhook ב-DSers

איך להגדיר ב-DSers:

  1. היכנס ל-DSers Dashboard
  2. לך ל: Settings → Webhooks
  3. לחץ על Add Webhook
  4. בחר ב-Order Status Updated או Tracking Available
  5. הדבק את ה-URL למעלה
  6. בחר ב-POST method
  7. שמור והפעל

מה יקרה אוטומטית:

  • כש-DSers יקבל מספר מעקב - הוא יישלח אוטומטית לחנות
  • המערכת תתאים את המספר להזמנה לפי CJ Order ID
  • המערכת תבדוק אוטומטית את סטטוס המשלוח כל 2 שעות
  • לקוחות יראו עדכון סטטוס בזמן אמת

הגדרות DSers Open API

איפה למצוא את המפתחות:

  1. היכנס ל-DSers Dashboard: https://www.dsers.com
  2. לך ל: Settings → Open API
  3. לחץ על Create App
  4. מלא שם אפליקציה (למשל: "VEXO Store")
  5. תקבל App Key ו-App Secret
  6. הדבק אותם כאן למטה

🔐 הזן את המפתחות:

סנכרון אוטומטי:

המערכת תסנכרן מספרי מעקב אוטומטית כל 30 דקות מה-API של DSers.

חיבור WooCommerce מזויף ל-DSers

למה זה עוזר?

DSers דורש חיבור חנות לפני שנותן גישה להגדרות Open API. הסקריפט הזה יוצר WooCommerce API מזויף ש-DSers יחשוב שזו חנות אמיתית. אחרי החיבור - תוכל לגשת ישירות להגדרות Open API!

🎭 פרטי חיבור מזויף:

שלבי החיבור:

  1. לחץ "צור פרטי חיבור מזויף" למעלה
  2. העתק את ה-3 שדות (Store URL, Key, Secret)
  3. ב-DSers בחר "WooCommerce" והדבק אותם
  4. DSers יחשוב שחיברת חנות! 🎉
  5. כעת תוכל לגשת ל-Settings → Open API
  6. צור App חדש וקבל את ה-App Key וה-App Secret
